background

background-color: liner-gradient(XXX);

第1引数に方向、第2引数に開始色、第３引数に終了色
下方向 => to bottom
上方向 => to top
左方向 => to left
右方向 => to right

終了色の始まる位置を指定できる 終了色の始まる位置を、第３引数に指定して、第４引数に終了色を指定する

斜めのグラデーション
開始位置を top left のように指定すると左上となり、to bottom right でも左上から、右下の意味になる
to で終わりの方向を指定して記述している場合は、それに揃えると保守につながる
また、斜め方向は具体的な角度も指定できる

degの値は
0deg => 下から上
45deg => 左下から右上
90deg => 左から右
180deg => 上から下

background-color: radial-gradient(XXX);

radial-gradientは円状に変化する色のグラデーションをつける
グラデーションで変化する範囲を指定できる
例えば、radial-gradient(中央色 20%, 外側色 70%)とすれば、中心から見て0〜20%は中央色で塗られ、
20〜70%がグラデーションに、70〜100%は外側色で塗られる
複数色の円形グラデーションをかける場合はlinear-gradientと書き方はほとんど同じ
色を変えたい順にカンマ区切りで色指定
例えば、3色なら中心から0〜40%で「中央の開始色 → 中間色」に、40〜100%で「中間色 → 終了色」にグラデーションがかかっている
画像がある場合
background: xxx-gradient(xxxx), url(xxx); と書くと画像を用いながらグラデシーション色もかけられる
url関数が先か後でxxx-gradientのレイヤーが決まる

background-position は CSS のプロパティで、それぞれの背景画像の初期位置を設定します。位置は background-origin で設定された位置レイヤーからの相対です
指定する値は位置を X軸 Y軸を指定し、

  • 引数1つなら、X軸を指定し、自動で2つめの引数は中央(center)となる
  • 引数2つなら、1つ目はX軸、2つ目がY軸
  • 引数3つなら、その値の指定方法によるので以下を参照する

topのみの指定

leftのみの指定

bottom right の指定

bottom 10% right 10% の指定

center の指定

center 20% の指定

左端から Xpx の指定

左端から Xpx, 上端から Ypx の指定

topから Xpx, rightから Ypx の指定
